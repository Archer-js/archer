Archer, the library that lets you control your data


You can make reusable web components: 
                
                    Archer.component("cool-button", {

                        template: `

                            <button style="background:blue;color:white;padding:16px 32px;border-radius:5px;">I'm a cool button</button>

                        `

                    });
and then use them like this: 
                
                    {{{cool-button}}}
Archer then turns it into this: 
                
                    <button style="background:blue;color:white;padding:16px 32px;border-radius:5px;">I'm a cool button</button>


You can also make variables: 
                
                    Archer.data.add("hello", "world!");
and write their value like this: 
                
                    {{hello}}
Archer then turns it into this: 
                
                    world!
